== Description ==


*Custom wp-admin / wp-login.php* is avery user friendly plugin that lets users safely modify the link of the Dashboard Login Page. It will not change any file name or even didn't modify any core file or even it will not rewrite any rule in your .htaccess and wp-config file. It only checks the URL and redirect the user to specific admin url. You can secure your admin dashboard area by using this plugin.

If you have any problem then you can let us know or you can drop us a mail on sharmakrishankant9@gmail.com

= Compatibility =


Requires WordPress 4.1 or higher.

It might not work with another plugin or hardcoded *wp-admnin/wp-login.php*. 

It might not Works with multisite. It might not work with Network Site


== Installation ==

1. Go to Plugins › Add New.
2. Search for *Hide wp-admin / wp-login.php*.
3. Look for this plugin, download and activate it.
4. The page will redirect you to the Custom Login Setting. Change your login url there.
5. You can also add the redirect URL like when user use the wrong url then you can move the user to 404 pr any other page.
5. You can change this option any time you want, just go back to Settings › Hide wp-admin / wp-login.php.
